# D15 反常配角驱动实战案例

## 案例一：信息差链

**项目**：都市异能文
**配角**：王管事（主角的助手，独立动机：想查清自己儿子的死因）
**推演过程**：
1. 主角视角：王管事最近几天总是请假，主角以为是家里有事
2. 独立事件（D15 方向）：王管事实际上去接触了另一个势力——他儿子生前任职的研究所
3. 碰撞点：他在研究所发现了一份关于主角的实验记录——但他没有告诉主角
4. 以「主角察觉到王管事的态度变了」的方式呈现

**效果**：读者知道王管事藏了什么，主角不知道。认知差（D7 叠加）持续 3-5 章后才引爆。

## 案例二：多米诺骨牌

**项目**：修仙文
**配角**：赵师弟（外门弟子，独立动机：想进入内门证明自己）
**推演过程**：
1. 主角视角：一切如常，赵师弟仍在修炼
2. 独立事件：赵师弟为了获取任务积分，接了清理妖兽的任务
3. 碰撞点：他在妖兽巢穴中触发了一个古老封印——释放了一头沉睡的妖兽
4. 主角数天后路过该区域时遭遇妖兽袭击

**效果**：配角的一个正常选择（接任务）引发了主角的危机，但配角完全不知情。主角被连累（被动牵连型碰撞）。

## 案例三：隐秘利益

**项目**：商战异能文
**配角**：林姑娘（主角的红颜知己，独立动机：想查清父亲失踪真相）
**推演过程**：
1. 主角视角：林姑娘说要去外地处理家族事务
2. 独立事件：她实际接触了主角正在调查的敌对公司的高管——有交易谈判
3. 碰撞点：她的接触在两个势力之间建立了隐秘联系通道——成了后续信息传递的桥梁
4. 主角后来发现「敌人的信息渠道中有林姑娘」的踪迹，产生怀疑

**效果**：林姑娘的独立动机与主角的利益产生冲突，但不是背叛——她只是有自己要办的事。配角的独立性和关系的真实性同时增强。

## 案例四：未知援助

**项目**：末世异能文
**配角**：瘸腿老张（营地铁匠，独立动机：保护自己收养的孤儿）
**推演过程**：
1. 主角视角：老张只是普通铁匠，偶尔修装备
2. 独立事件：有势力想抓老张的孤儿，老张用隐藏手段赶走了他们
3. 碰撞点：老张暴露了自己的隐藏实力，被那个势力盯上——同时他挡掉的这批人原本下一步计划绑架主角
4. 主角根本不知道这件事，但营地的安全局势已经变了

**效果**：主角完全不知道配角做了什么。这向读者传递「这个世界不会因为主角不知道就停止运转」的感觉。

## 选择配对的规则

| 条件 | 推荐 | 原因 |
|------|------|------|
| independent_goal 近期未推进 | 优先选 | 配角的动机需要被赋予重量 |
| 5 章以上未重点描写 | 优先选 | 制造的"惊喜回归"感最强 |
| 配角当前就在主角身边 | 考虑用信息差链 | 在身边时信息差最焦虑 |
| 配角当前不在主角身边 | 考虑用多米诺骨牌 | 未知空间制造悬念 |
| 配角有隐藏身份/隐藏实力线索 | 考虑用隐秘利益/未知援助 | 配合隐藏设定回收 |
| 配角的功能定位是 emotion_regulator | 谨慎用 | 情绪调节型配角搞大事会破坏读者预期 |
