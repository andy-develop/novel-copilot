# 叙事纪律与读者情绪管理

## 9.1 叙事克制（全章通用）

**法则一：300字入局**
- 每章前300字必须进入冲突/行动/紧张感。禁止环境描写、背景回顾、情绪独白开篇。
- 写后自检：删掉前300字，后面不受影响 → 那些字是废话。

**法则二：背景融入，不独白式罗列**
- 禁止超过200字的纯设定/背景说明段落。
- "纯设定段落"判定标准（3条件同时满足）：①没有角色在行动 ②没有冲突或紧张感 ③信息流向是"作者→读者"
- 技巧：用"主角需要用到某个设定"的场景引入，而非"作者想告诉读者"的场景。

**法则三：目标驱动，不慢热**
- 主角从第一章起必须有明确的近期目标（不是远大理想）。
- 3章内没有主动行动 → 下章必须补上主动目标。

**写后叙事自检**（每章）：
```
□ 前300字是否直接进入了剧情？
□ 有无超过200字的纯设定/背景说明段落？
□ 主角在本章是否有主动行动或明确目标？
□ 本章内容删掉后，主线是否受损？不受损=可删
```

## 9.2 断章与高潮控制

**纪律一：正常断章——在该断的地方断**
- 好章的结尾不是"事情解决了"，而是"事情正到最关键处"。
- 断章黄金位置：决策即将做出之前 / 真相即将揭示之前 / 冲突即将爆发之前 / 不安新信息出现 / 角色正要踏入危险不知情

**纪律二：大高潮分拆法则**
- 高潮内容超过5000字 → 必须分拆为2-3章。
- 分拆节奏：高潮前夜（选择即将做出时断）→ 高潮爆发（最紧要关头断）→ 高潮余波（结果揭示+新悬念，可收束）

**纪律三：断章自检**
- 读者看到最后一行感觉"好吧"而非"然后呢？" → 断章失败
- 好的断章让读者带着1-2个未解的问题离开

## 9.3 早期情绪保护（前10章）

**核心原则：前10章不虐主**

红线：
- ❌ 主角被彻底击败且无法反击
- ❌ 被剥夺核心能力/资源且短期无法恢复
- ❌ 被严重羞辱且无反击机会
- ❌ 重要盟友死亡（除非死亡成为主角驱动目标）
- ⚠️ 连续2章只被动承受 → 下章必须主动行动（第3章被动=一票否决）

**"能动性"原则**：前10章主动行动占比应≥60%。主动行动=主角主动发起/选择/承担风险；被动反应=被打/被骗/被追。

**第3-10章特殊评分**：方向含爽点 → 翻页驱动+1/情感+0.5；连续2章零爽点 → 翻页驱动-1；前10章爽度密度加倍（每2章≥1中爽点）。

**第10章情绪审查**：
```
□ 主角是否有至少3次主动出击？
□ 是否遭遇过"无还手之力"的严重打压？
□ 读者是"在乎"主角还是"同情"主角？（在乎✅ 同情❌）
□ 主角是否从被动变主动了？
□ 整体情绪走向是否"低开高走"？
```

## 9.4 通俗易懂原则（全书通用）

**核心问题**：AI 写作容易陷入"解释癖"——为了让设定看起来完整，堆砌大段术语、功法原理、势力层级介绍。网文读者不需要这些，需要的是**在行动中感知设定**。

**三不写**（否决级）：
- 🚫 **不写纯设定段落**：连续超过150字纯阐述世界观/功法/势力/历史，无角色行动穿插 → 违规
- 🚫 **不写名词轰炸**：单段内首次出现≥3个生造术语（功法名/地名/势力名/境界名）且无上下文消化 → 违规
- 🚫 **不写体系说明文**：用旁白把一套完整体系（修行体系/规则体系/力量等级表）一次性讲完 → 违规

**三要写**（执行级）：
- ✅ **设定融入行动**：设定信息在场景中"露出来"——角色做出选择时顺便带出规则，敌人展示能力时带出体系，环境描写带出地理，**而不是让叙事停下来讲解**
- ✅ **名词即生即用**：新术语出现后，300字内必须看到它在行动中发生作用（打斗、对话、推理、抉择），否则读者记不住
- ✅ **优先级穿透**：当【设定完整展示】与【叙事流畅性】冲突时，**叙事流畅性永远优先**。如果为了保持节奏而需要牺牲某个设定细节，写章后在 threads.yaml 记一笔即可

**自检清单**（每章自查）：
```yaml
□ 本章是否有超过150字的纯说明段落？（有→重写，拆散融入行动）
□ 本章首次出场的新术语是否≤3个？（>3→削减或推迟介绍）
□ 每一个新术语是否在300字内有行动证实？（否→补一个使用场景）
□ 删掉某个设定说明，故事节奏是否更流畅？（流畅=说明多余）
□ 本章可以被「对设定一无所知」的读者完全看懂吗？
```
