# 叙事纪律与读者情绪管理

## 9.1 叙事克制（全章通用）

**法则一：300字入局**
- 每章前300字必须进入冲突/行动/紧张感。禁止环境描写、背景回顾、情绪独白开篇。
- 写后自检：删掉前300字，后面不受影响 → 那些字是废话。

**法则二：背景融入，不独白式罗列**
- 禁止超过200字的纯设定/背景说明段落。
- "纯设定段落"判定标准（3条件同时满足）：①没有角色在行动 ②没有冲突或紧张感 ③信息流向是"作者→读者"
- 技巧：用"主角需要用到某个设定"的场景引入，而非"作者想告诉读者"的场景。

**法则三：目标驱动，不慢热**
- 主角从第一章起必须有明确的近期目标（不是远大理想）。
- 3章内没有主动行动 → 下章必须补上主动目标。

**写后叙事自检**（每章）：
```
□ 前300字是否直接进入了剧情？
□ 有无超过200字的纯设定/背景说明段落？
□ 主角在本章是否有主动行动或明确目标？
□ 本章内容删掉后，主线是否受损？不受损=可删
```

## 9.2 断章与高潮控制

**纪律一：正常断章——在该断的地方断**
- 好章的结尾不是"事情解决了"，而是"事情正到最关键处"。
- 断章黄金位置：决策即将做出之前 / 真相即将揭示之前 / 冲突即将爆发之前 / 不安新信息出现 / 角色正要踏入危险不知情

**纪律二：大高潮分拆法则**
- 高潮内容超过5000字 → 必须分拆为2-3章。
- 分拆节奏：高潮前夜（选择即将做出时断）→ 高潮爆发（最紧要关头断）→ 高潮余波（结果揭示+新悬念，可收束）

**纪律三：断章自检**
- 读者看到最后一行感觉"好吧"而非"然后呢？" → 断章失败
- 好的断章让读者带着1-2个未解的问题离开

## 9.3 早期情绪保护（前10章）

**核心原则：前10章不虐主**

红线：
- ❌ 主角被彻底击败且无法反击
- ❌ 被剥夺核心能力/资源且短期无法恢复
- ❌ 被严重羞辱且无反击机会
- ❌ 重要盟友死亡（除非死亡成为主角驱动目标）
- ⚠️ 连续2章只被动承受 → 下章必须主动行动（第3章被动=一票否决）

**"能动性"原则**：前10章主动行动占比应≥60%。主动行动=主角主动发起/选择/承担风险；被动反应=被打/被骗/被追。

**第3-10章特殊评分**：方向含爽点 → 翻页驱动+1/情感+0.5；连续2章零爽点 → 翻页驱动-1；前10章爽度密度加倍（每2章≥1中爽点）。

**第10章情绪审查**：
```
□ 主角是否有至少3次主动出击？
□ 是否遭遇过"无还手之力"的严重打压？
□ 读者是"在乎"主角还是"同情"主角？（在乎✅ 同情❌）
□ 主角是否从被动变主动了？
□ 整体情绪走向是否"低开高走"？
```
