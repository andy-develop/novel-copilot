# 章节解剖引擎 — 完整扩容技法

> 配合 SKILL.md 模块十二使用。解决「AI 把 500 字情节节点扩展成 2000 字完整章节」的核心难题。

## 一、八元素详解（含中文网文示例）

### A·入局钩子（100-200字）

**黄金开篇类型**：

| 类型 | 示例 |
|------|------|
| 半途入局 | 「刀已经架在他脖子上了。」（跳过铺垫，直接冲突） |
| 感官冲击 | 「血腥味。浓得化不开的血腥味。」 |
| 对话直入 | 「『你儿子在我手上。』」 |
| 异常发现 | 「账本上少了一笔。不多不少，刚好够他死。」 |

**禁止的开篇**：天气描写、角色醒来、背景回顾、环境铺陈。

### B·行动序列（400-600字）

核心事件。写「发生了什么事」，不写「这意味着什么」。
- 战斗：一拳一脚都是画面，禁止「大战三百回合」式概括
- 谈判：每句话推进信息或施加压力
- 发现：展示发现过程，不只展示结果

### C·反应段落（200-300字）

三重反应（缺一不可）：
1. **身体**：手在抖/背贴墙/腿发软/呼吸不畅
2. **情绪**：通过感官展示（血涌上脸/胃在下坠/眼前发白）
3. **策略**：「这意味着接下来他必须——」→ 引出下一段行动

### D·后果涟漪（200-300字）

新问题从旧答案中诞生：
- 「他赢了。但赢了的他在宗门里更危险了。」
- 「消息传开需要时间。但有些人比消息快。」

### E·配角时刻（200-300字）

**每章至少1个配角的独立时刻**。配角不光是主角的助手/敌人，他们有自己关心的事：
- 师妹在深夜独自练剑——她在怕什么？
- 老掌柜关门前多看了一眼神龛——他在求什么？

### F·世界质感（100-200字）

融入行动，非独立段落：
- ❌ 独立段落：「这个世界有三大宗门。青云宗位于东方……」
- ✅ 融入行动：「他路过青云宗的山门时，脚步本能地放轻了。十七年的习惯，不是说改就能改的。」

### G·子线推进（200-300字）

每章至少1条副线向前移动一步。哪怕只是：
- 远处一场和主角无关的战斗结果传来
- 某个人物收到一封信后表情变了
- 一个此前埋下的伏笔被激活了一小步

### H·断章钩子（100-200字）

**五类钩子轮换**（不可连3次同类型）：

| 类型 | 示例 |
|------|------|
| 悬念钩子 | 门开了。门外的不是人。 |
| 反转钩子 | 「你找的那个人——十年前就死了。」 |
| 力量时刻 | 系统提示亮起的那一刻，他知道游戏规则改了。 |
| 情感重击 | 她没有哭。只是把碗筷多摆了一副。 |
| 追问钩子 | 但如果他是对的呢？如果所有人都错了呢？ |

## 二、六大扩容技法

### 技法1：反应场景（150-300字）

每次重大行动后，强制插入主角的三重反应。这是**最容易被AI跳过的内容**——AI倾向于直接跳到下一个行动。

### 技法2：旁观者视角（150-250字）

通过另一角色的眼睛重新看这个行动。这个人看到了什么主角没看到的？
「老陈在角落里看着。年轻人的招式他见过。五十年前。在另一个人的身上。」

### 技法3：感官铺层（100字）

每次地点切换强制触发感官锚点。最少2种感官（不能全是视觉）。

### 技法4：微型闪回（100-150字）

当前事件触发的一段记忆。规则：
- 不超过150字
- 必须被当前事件明确触发（不是作者想说就切）
- 每章最多1次

### 技法5：势力涟漪（150-200字）

主角的行动如何影响其他群体：
「他在擂台上只用了三招。但那天晚上，至少五个势力修改了对他的评估报告。」

### 技法6：代价对话（200-300字）

角色讨论这个行动**意味着什么**，而非**发生了什么**：
- ❌ 「你把那个长老杀了。」「对。」「怎么杀的？」「一剑。」
- ✅ 「你杀了一个长老。知道这意味着什么吗？」「意味着少了一个长老。」「意味着你得补上。」

## 三、写前扩展自检清单

```
□ 本章的单一核心行动是？（1句话）
  → 行动开始前发生了什么？（入局钩子）
  → 行动后，主角的身体/情绪/策略？（反应段落）
  → 还有谁被影响？怎么表现？（后果涟漪）

□ 哪个配角在本章有独立时刻？（每章至少1个，强制轮换配角）
□ 哪条副线推进了一步？（哪怕很小——一个信息/一个变化/一个靠近的威胁）
□ 这个场景闻起来/听起来/摸起来是什么？（至少2种非视觉感官）
□ 章末留下什么追问？（读者不可能看完这章不去看下一章）
□ 有没有一段≤150字的记忆能加深主角的当下选择？
```

### 章节薄弱预警信号

- 核心行动是「他们谈话和计划」→ **加入中断**（谈话中闯入消息/发现/敌人）
- 只有1个角色 → **立刻加入第2个**（哪怕只是路过、偷听、远远看着）
- 全程1个场景 → **移1个场景到别处**
- 零身体状态变化 → **加入疲惫/受伤/温度/饥饿**
- 零配角时刻 → **下一章必须轮换到一个超过3章没出现的配角**

## 四、章节密度清单

每章写完后逐项检查：

```
□ 2+场景（或1个场景发生意义变化）
□ 2+角色有意义互动（不只是「在场」——有对话/冲突/信息交换）
□ 1+感官变化（声音变了/气味变了/温度变了）
□ 1+新信息（揭示给主角或读者，非重复已知信息）
□ 1+决策做出（主角选择了什么，后果暗示）
```

**缺失3/5 → 章节太薄，无论字数多少，必须回补。**
